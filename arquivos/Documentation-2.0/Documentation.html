1. Preparing - top

Make sure you have the following:
  1. PHP 5.3+
  2. MySQLi Extension

2. Installation Process - top

  1. Upload the files from the "Script" folder on the FTP server (usually public_html folder when installing it on a domain).
  2. Set the CHMOD to 777 or 775 (depending on the server configuration) to the following folders: 
    admin/mega-cache
mega-cache
uploads
  3. Goto URL web site http://url.com/setup/install.php Click button Begin Install :)
  4. Delete folder http://url.com/setup
